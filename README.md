# ⚗️ Physical Units Converter

### 🧪 Инструмент для быстрой и точной конвертации физических величин с современным GUI

---

## 🛠️ Стек технологий проекта

**Основной стек:**

![Python](https://img.shields.io/badge/Python-3776AB?style=for-the-badge&logo=python&logoColor=white)
![Tkinter](https://img.shields.io/badge/Tkinter-gray?style=for-the-badge&logo=python&logoColor=white)

**Дизайн и логика:**
- **UI:** Кастомная темная тема (палитра Catppuccin Mocha).
- **Математика:** Поддержка стандартных коэффициентов и формул смещения для температурных шкал.
- **UX:** Обработка ошибок ввода и адаптивное форматирование чисел (Scientific Notation).

---

## ✨ Основные возможности

- 📐 **6 категорий измерений:** Длина, Масса, Температура, Скорость, Энергия и Давление.
- ⚡ **Мгновенный результат:** Конвертация происходит в реальном времени при вводе.
- 🔄 **Smart Swap:** Быстрая смена направления конвертации одной кнопкой.
- 🔢 **Умный вывод:** Автоматическое сокращение лишних нулей и поддержка экспоненциальной записи для сверхмалых/больших величин.
- 🎨 **Modern Design:** Интерфейс в стиле современных IDE, не нагружающий глаза.
